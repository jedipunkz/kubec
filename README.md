# kubec

## Description

kubec is kubernetes cli for my own and learning go.

## Sub-Commands

### list deployments

```bash
$ kubec list deployments [-n <namespace>]
```

### list pods

```bash
$ kubec list pods [-n <namespace>]
```

### list namespaces

```bash
$ kubec list namespaces
```
